# 🌱 AgriMind Dashboard - AI Crop Care Assistant

**AgriMind** is a modern, AI-powered web application designed for farmers, gardeners, and agri-enthusiasts in India. It helps users manage plants, diagnose diseases, get personalized crop recommendations, and access instant AI farming advice—all in one intelligent dashboard.

## 🚀 Features

### 1. **Plant Management Hub**
*   **Add Plants**: Register crops by name, type, and city for personalized tracking
*   **Your Plant List**: View and manage your complete plant portfolio locally

### 2. **AI-Powered Farming Assistant** 
*   **Smart Chatbot**: Instant answers on watering, soil, fertilizers, pests, and sunlight
*   **Context-Aware**: Understands plant names and gives specific care tips

### 3. **Disease Diagnosis**
*   **Image Upload**: Detect plant diseases (upload photos for analysis)
*   **Instant Results**: AI-powered disease identification and treatment

### 4. **Smart Crop Recommendations**
*   **Indian soil types** (Alluvial, Black, Red, etc.)
*   **Soil nutrients** (Nitrogen, Phosphorus, Potassium)
*   **Provides crop** suggestions tailored to Indian agriculture conditions

### 5. **Interactive Dashboard**
*   **Glassmorphism UI**: Modern, translucent design with smooth animations
*   **Floating Widgets**: AI chat and language switcher always accessible
*   **Responsive**: Perfect on desktop, tablet, and mobile

## 🛠️ Technology Stack

*   **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript (No frameworks)
*   **Styling**: Custom glassmorphism design system with gradient animations
*   **Storage**: LocalStorage for plant tracking (offline-first)
*   **Icons**: Font Awesome + Emoji
*   **Animations**: Pure CSS keyframes (gradient shift, float, pop effects)

## 💻 How to Run Locally

1. **Clone/Download repository:**
   ```bash
   git clone https://github.com/mekanH-c/Agriminds.git
   cd Agriminds
   ```

2. **Open in browser:**
   ```
   Open index.html directly (no server needed!)
   ```

3. **Start using:**
   - Click feature cards to open sections
   - Add plants (saved locally)
   - Chat with AI 🌿 (bottom-right)
   - Switch language 🌐 (bottom-left)

## 🧪 Quick Test Flow

1. **Click "Add Plants"** → Fill form → See your plant list
2. **Click AI chat (🌿)** → Ask "how to water tomato?"
3. **Language switch (🌐)** → Try हिंदी translation
